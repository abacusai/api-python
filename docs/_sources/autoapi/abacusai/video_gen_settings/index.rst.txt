   Video generation settings

   :param client: An authenticated API Client instance
   :type client: ApiClient
   :param videoType: Dropdown for type of video (text_to_video, image_to_video, lip_sync).
   :type videoType: dict
   :param modelsByType: Maps each video type to the list of applicable model keys.
   :type modelsByType: dict
   :param imageFieldsByModel: Maps each model to the list of image input field names.
   :type imageFieldsByModel: dict
   :param mediaCapabilities: Maps each model to its accepted media types and input slots (derived from settings).
   :type mediaCapabilities: dict
   :param settings: The settings for each model.
   :type settings: dict
   :param warnings: The warnings for each model.
   :type warnings: dict
   :param descriptions: The descriptions for each model.
   :type descriptions: dict
   :param audioModes: Maps models without a generate_audio toggle to a fixed audio behaviour ('alwaysOn'/'none').
   :type audioModes: dict
   :param model: Dropdown for models available for video generation.
   :type model: VideoGenModel
